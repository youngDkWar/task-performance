1. Добавил для скриптов атрибуты async и defer. Скорость закрузки скрипта contacts.js упала с ~800ms до ~600ms
2. Удалил свойство scroll-snap-type для "contacts". Прокручивание стало значительно плавнее. 
3. В contacts.js переработал систему отрисовки видимого блока с контактами. Теперь блок контактов изначально содержит 50 элементов, 
вместо 50 000. Каждые 3 прокрута блока увеличивают базу на 50 контактов. Таким образом, даже при минимальном масштабе страницы, за один
прокрут появляется только 7 новых элементов, поэтому 50 новых вполне достачточно для любого размера экрана. Это ускрояет отрисовку В
тысячу раз.
4. Убрал свойства, не изменяющие ничего на странице в style.css и удалил все лишние символы (пробелы, переносы) чтобы сократить размер файла.
